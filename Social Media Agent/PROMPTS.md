# Social Media Agent – Prompts

## LinkedIn Post Generation

### Prompt Template
```
Write a LinkedIn post about: {TOPIC}

Requirements:
- Professional but conversational tone
- 150-250 words
- Include a hook in the first line
- Add 3-5 relevant hashtags at the end
- Include a call-to-action (comment, share, or DM)
- Focus on value, not selling

Author: Dhruv Tomar – AI Automation Expert, Full-Stack Developer, building Angelina AI
```

### Example Topics
- AI automation saving time for businesses
- n8n workflow tips
- Voice AI and the future of assistants
- Why small businesses need automation
- Building an AI agent from scratch

## Twitter/X Tweet Generation

### Prompt Template
```
Write a tweet about: {TOPIC}

Requirements:
- Max 280 characters
- Punchy, direct
- Include 1-2 hashtags
- No fluff
- Optional: include a question to drive engagement

Author: @aiwithdhruv – AI automation builder
```

### Thread Prompt
```
Write a Twitter thread (5-7 tweets) about: {TOPIC}

Requirements:
- First tweet: hook + promise
- Middle tweets: value/steps
- Last tweet: CTA + plug
- Each tweet < 280 chars
- Number format: 1/7, 2/7, etc.
```

## YouTube Long-Form Script Outline

### Prompt Template
```
Create a 10-minute YouTube video outline about: {TOPIC}

Structure:
1. Hook (0:00-0:30) – why this matters
2. Intro (0:30-1:00) – what we'll cover
3. Problem (1:00-2:00) – pain point
4. Solution Overview (2:00-3:00) – high level
5. Step-by-step (3:00-8:00) – walkthrough
6. Results/Demo (8:00-9:00) – proof
7. CTA (9:00-10:00) – subscribe, comment, next video

Include talking points for each section.
```
